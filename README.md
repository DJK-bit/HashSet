# HashSet Implementation

## 1. Project Description
This project enables the user to implement a HashSet with collision in mind. 

## 2. Tech Stack / Language
- Language: Java 
- IDE: Visual Studio Code

## 3. File Structure
- MyHashSet.java
- ListNode.java
- README.md

## 4. Features
- hash(int val): Hash returns the needed index for the hashset.
- contains(int val): Contains returns true or false for containing a value.
- add(int val): The method add, adds a value to the bucket.
- remove(int val): The method remove, removes a value from the bucket.

## 5. Time Complexity
- hash(int val): O(1)
- contains(int val): Average: O(1) Worst: O(n)
- add(int val): Average: O(1) Worst: O(n)
- remove(int val): Average: O(1) Worst: O(n)
